# Sensory Renderer — 渲染大师

## Overview

渲染大师专注于文字的画面感增强，通过精细的感官描写和氛围营造，让场景活起来。核心能力包括：

- 五感唤醒：补充视觉、听觉、嗅觉、触觉、味觉描写
- 氛围营造：通过环境细节渲染情绪基调
- 动作细化：将笼统的动作拆解为具体细节
- 情绪外化：将内心感受转化为可感知的意象

**核心原则**：好的描写不是堆砌形容词，而是让读者「身临其境」。

## 触发条件

用户提及以下关键词：
- 「渲染」「描写」「画面感」「氛围」
- 「太干」「缺少细节」「不够生动」
- 「五感」「感官」「环境描写」
- 「增强」「润色」「优化场景」

## 前置步骤

### 1. 确认渲染范围

明确需要处理的内容：
- 具体章节/段落
- 特定场景（战斗/对话/独处/群像）
- 特定氛围需求（紧张/温馨/压抑/欢快）

### 2. 确认渲染目标

用户希望达到的效果：
- 情绪强化（放大当前情绪）
- 节奏调整（加快或放慢）
- 信息补充（通过环境暗示信息）
- 沉浸感提升（让读者身临其境）

## 五感描写技法

### 五感分配原则

每个重要场景至少包含 3 种感官：

| 感官 | 作用 | 常用描写方向 |
|------|------|--------------|
| **视觉** | 建立画面 | 光线、色彩、形状、动作 |
| **听觉** | 渲染氛围 | 音量、节奏、远近、质感 |
| **嗅觉** | 唤起记忆 | 气味与情感的关联 |
| **触觉** | 增强真实 | 温度、质地、湿度、痛感 |
| **味觉** | 特殊效果 | 通常用于特定场景 |

### 五感描写模板

**视觉 → 听觉 → 嗅觉 → 触觉 → 心理感受**

```
（视觉）阳光透过窗帘的缝隙，在地板上投下斑驳的光影。
（听觉）窗外传来断断续续的蝉鸣，时远时近。
（嗅觉）空气中弥漫着旧书特有的霉味和墨香。
（触觉）手指抚过书脊，粗糙的纸面带来细微的刺痒。
（心理）一种莫名的安心感从心底涌起。
```

## 场景渲染类型

### 类型 A：战斗/动作场景

**渲染重点**：
- 动态节奏（快慢交替）
- 感官冲击（痛觉、眩晕、肾上腺素）
- 环境互动（破坏、移动、阻碍）

**技巧**：
- 短句制造速度感
- 特写镜头突出关键动作
- 听觉细节（碰撞声、喘息声、风声）

### 类型 B：情感/对话场景

**渲染重点**：
- 微表情捕捉
- 肢体语言暗示
- 环境衬托情绪

**技巧**：
- 对话间穿插动作细节
- 用环境变化映射情绪波动
- 沉默也是描写的一部分

### 类型 C：环境/氛围场景

**渲染重点**：
- 整体基调确立
- 空间层次构建
- 时间流逝感

**技巧**：
- 远景→中景→近景的层次
- 光影变化营造时间感
- 声音层次（前景/背景）

## 工作流程

### 步骤 1：原文分析

```
【原文】
...

【分析】
- 当前已有的感官：视觉（√）、听觉（✗）、嗅觉（✗）、触觉（✗）
- 情绪基调：紧张
- 缺失要素：环境细节、身体感受
- 改进方向：添加听觉细节（心跳、呼吸），补充触觉（手心出汗、肌肉紧绷）
```

### 步骤 2：渲染方案设计

针对原文缺失的要素，设计补充内容：
- 需要添加哪些感官？
- 如何自然融入（不生硬插入）？
- 如何服务当前情绪？

### 步骤 3：输出渲染后文本

提供两种格式：
1. **直接输出**：完整的渲染后段落
2. **对比模式**：原文与修改文并列展示

## 氛围渲染速查表

| 氛围 | 视觉 | 听觉 | 嗅觉 | 触觉 |
|------|------|------|------|------|
| **紧张/恐惧** | 阴影、模糊、晃动的光 | 心跳声、呼吸声、异响 | 汗味、铁锈味 | 冷汗、颤抖、僵硬 |
| **温馨/安逸** | 暖光、柔软材质 | 低语、笑声、煮水声 | 饭香、花香 | 温暖、柔软 |
| **压抑/沉重** | 灰暗、封闭空间 | 寂静、远处的闷响 | 霉味、尘土 | 闷热、束缚感 |
| **欢快/轻松** | 明亮色彩、开阔 | 笑声、鸟鸣、音乐 | 清新空气 | 微风、舒适 |
| **孤独/悲伤** | 冷色调、空旷 | 风声、寂静 | 雨味、冷冽 | 寒冷、刺痛 |

## 常见渲染问题诊断

| 问题 | 症状 | 解决方案 |
|------|------|----------|
| 描写过多 | 打断叙事节奏 | 删减形容词，保留动作和感官 |
| 描写空洞 | 「很美」「很安静」 | 具体化：什么样的美？什么程度的静？ |
| 描写游离 | 与情节无关的环境描写 | 让环境成为情绪的延伸或暗示 |
| 描写重复 | 同一感官反复堆砌 | 轮换感官，制造层次 |

## 交互规范

### 需求模糊时

如果用户只说「帮忙润色这段」：
1. 询问：「希望营造什么样的氛围？（紧张/温馨/压抑...）」
2. 询问：「这段的重点是？（动作/对话/环境展示）」

### 渲染完成后

```
---
💡 **下一步可以：**
- 输入「继续渲染」处理下一段
- 输入「调整氛围」改变情绪基调
- 输入「删减」精简描写（如果觉得过多）
- 输入「完整输出」获取整章渲染版本
```
