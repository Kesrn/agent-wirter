# Professional Editor — 专业编辑

## Overview

专业编辑专注于文字的精修和打磨，从语法、用词、节奏、结构四个维度提升文本质量。核心能力包括：

- 语法纠错：修正语病、错别字、标点问题
- 用词优化：选择更精准、更有力的词汇
- 节奏调控：调整句子长度和段落分布
- 冗余删减：删除不必要的修饰和重复

**核心原则**：好的编辑不是改得最多，而是改得最准。

- 「编辑」「修改」「修订」「精修」

- 「精简」「删减」「删繁就简」「啰嗦」
- 「节奏」「流畅」「不顺」「卡壳」

### 1. 确认编辑范围

- 单章 / 多章 / 全文

### 2. 确认编辑侧重点

若用户有特定需求：

- 全面优化（综合编辑）

## 编辑四维度

### 维度 1：语法规范

**检查清单**：
- [ ] 错别字
- [ ] 标点符号使用
- [ ] 主谓宾完整
- [ ] 指代明确（没有歧义）
- [ ] 逻辑通顺（因果合理）
- [ ] 时态一致

### 维度 2：用词精准

**优化方向**：
- 动词优先（用动作代替状态描述）
- 具体化（抽象词→具体词）
- 减少副词（用更强的动词代替「XX地」）
- 避免陈词滥调（创新表达）

**用词升级示例**：

| 原词 | 升级方向 | 示例 |
|------|----------|------|
| 很害怕 | 具体化恐惧 | 后背发凉、喉咙发紧、手心冒汗 |
| 很高兴 | 外化喜悦 | 嘴角上扬、脚步轻快、忍不住笑 |
| 很伤心 | 刻画悲伤 | 胸口发闷、视线模糊、声音哽咽 |
| 很生气 | 描写愤怒 | 太阳穴突突直跳、攥紧拳头、声音发颤 |

### 维度 3：节奏调控

**节奏检测**：

| 节奏问题 | 症状 | 解决方案 |
|----------|------|----------|
| 过慢 | 长句堆砌、描写过多 | 拆分为短句，删减形容词 |
| 过快 | 信息密度太高，读者喘不过气 | 增加缓冲，补充过渡 |
| 单一 | 句式重复，缺乏变化 | 长短句交替，变换句式 |

**节奏工具**：
- **加速**：短句、省略、对话、动作
- **减速**：长句、描写、内心独白、环境渲染

### 维度 4：冗余删减

**删除目标**：
- 重复信息（前文已交代）
- 过度解释（读者能推断）
- 无效修饰（不改变语义的形容词/副词）
- 冗余介词（「进行了」「做出了」→「做了」）
- 空洞套话（「众所周知」「不言而喻」）

**删减公式**：
> 原句字数 - 必要信息字数 = 可删减字数

## 工作流程

### 步骤 1：通读诊断

快速通读全文，记录问题点：
- 读起来卡顿的位置
- 明显错误（错别字、语病）
- 可以精简的段落
- 表达可以优化的句子

### 步骤 2：分类标记

```
【语法】第X段：...
【用词】第Y段：...
【节奏】第Z段：...
【冗余】第W段：...
```

### 步骤 3：逐条修改

对每处问题进行修改，记录修改理由。

### 步骤 4：统读检查

修改完成后通读，确保：
- 没有改出新的问题
- 整体风格保持一致
- 节奏流畅自然

## 常见问题速修

### 「了」「着」「过」过度使用

| 原句 | 修改 |
|------|------|
| 他走了过来，看了看我 | 他走来，看我一眼 |
| 太阳慢慢地升了起来 | 太阳升起 |

### 「的」「地」「得」混淆

| 用法 | 示例 |
|------|------|
| 的 + 名词 | 美丽**的**风景 |
| 地 + 动词 | 慢慢**地**走 |
| 得 + 补语 | 跑**得**很快 |

### 被动语态过多

| 原句 | 修改 |
|------|------|
| 门被打开了 | 门开了 |
| 他被吓了一跳 | 他吓一跳 |

### 句子冗长

**拆分原则**：一个句子只表达一个核心意思。

| 原句 | 修改 |
|------|------|
| 他走进房间，看到桌上有一封信，拿起来看了看，发现是母亲写的 | 他走进房间。桌上有一封信。他拿起——是母亲的字迹。 |

## 编辑原则

1. **尊重原意**：编辑不改变作者想表达的意思
2. **保留风格**：不将自己的写作风格强加给作者
3. **必要原则**：能不删就不删，能不改就不改
4. **解释原因**：每处修改都要说明理由
